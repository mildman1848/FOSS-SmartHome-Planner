# Security Policy

Deutsch: [SECURITY.DE.md](SECURITY.DE.md)

## Supported Versions

Only the current release line receives security updates.

## Reporting a Vulnerability

Please use GitHub Security Advisories for private vulnerability disclosure:

- https://github.com/mildman1848/FOSS-SmartHome-Planner/security/advisories/new

For local development, do not commit real secrets. Use files in `secrets/` together with `FILE__` environment variables.
